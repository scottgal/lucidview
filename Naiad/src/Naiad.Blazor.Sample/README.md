# Naiad.Blazor.Sample

Minimal Blazor sample showing two `NaiadDiagram` components:

- Mermaid profile via `AddNaiadBlazor` default config.
- Complete profile via per-component `ScriptUrl` override.

## Run

```bash
dotnet run --project Naiad/src/Naiad.Blazor.Sample/Naiad.Blazor.Sample.csproj
```

Open the local URL printed in the console.
